História do HTML


O HTML foi inventado pelo físico britânico Tim Berners-Lee em 1991, no centro de pesquisas CERN, na Suíça. Criado para facilitar o compartilhamento de documentos científicos entre pesquisadores, o sistema baseava-se em hipertexto (links) para conectar informações. A primeira versão possuía 18 tags e fundamentou a World Wide Web (WWW)

Principais Marcos da Evolução do HTML:
1991 (Origem): Primeira versão criada por Tim Berners-Lee com 18 tags básicas.,
1993 (Definição): A IETF (Internet Engineering Task Force) publica a primeira proposta de especificação HTML.,
1995 (HTML 2.0): Lançado com recursos essenciais como formulários e imagens.,
1997-1999 (HTML 3.2 e 4.01): Introdução de tabelas, melhorias de multimídia e regras mais rigorosas.,
2014 (HTML5): O maior upgrade, introduzindo tags semântica.




HTML (HyperText Markup Language) é a linguagem padrão para estruturar páginas web,
definindo elementos como títulos, parágrafos, imagens e links.
Sua estrutura base utiliza tags (html``head``body)
para organizar o conteúdo,
que os navegadores interpretam para exibir o site.
